# Bateria de Desafios 25

Esta Bateria contém apenas um desafio, mas você terá que pesquisar bastante =)

Seu desafio desta vez é converter o CSS do Quiz dos Videogames para `styled-components`, a lib CSS-in-JS (1) mais popular atualmente (2). 

Embora o uso Tailwind venha crescendo de forma exponencial (3), as chances de você encontrar projetos que usam `styled-components` também é muito alta (4 e 5). Por isso, é importante conhecer essa abordagem de estilização de projetos React. 

A versão exata do styled-components que instalei para implementar este desafio é a 6.1.8.

Para ter uma melhor experiência de desenvolvimento, recomendo que use a extensão oficial do `styled-components` para VSCode (6).
