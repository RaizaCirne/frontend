# Bateria de Desafios 31

Todos os desafios desta bateria são relacionados à aplicação `game-reviews`.

Lembre-se: se não foi explicado "como" fazer, descobrir é parte do desafio.

---

## Desafio 1

Dentro do componente Home, crie um alert que é executado quando a página Home é carregada no navegador. 

O alert deve exibir a mensagem "Bem-vindo(a)!".

---

## Desafio 2

Remova o alert da página Home e toda a implementação de código relacionada ao desafio acima. 

Deixe o arquivo do componente Home apenas com a declaração e exportação do componente Home. 

Rode a aplicação no server de produção. 

O objetivo deste desafio não é fazer deploy para produção. O deploy será feito depois. 

Este é um desafio que checa apenas se você sabe o que é necessário fazer para levantar o servidor de produção. 

Caso ainda não esteja totalmente claro o papel do servidor de produção, não se preocupe. Isso ficará mais claro mais à frente, quando fizermos deploy da aplicação. 

Após fazer o que é necessário para rodar o server de produção, derrube-o e execute o dev server. 

---

## Desafio 3

Implemente os links de navegação para as páginas Início, Sobre e Análises. 

Use a forma recomendada pelo Next.js para fazer isto. 

---

## Desafio 4

Visualize o prefetching dos links de navegação implementados no desafio acima.
