# Bateria de Desafios 27

Lembre-se: se não foi explicado "como" fazer, descobrir é parte do desafio.

---

Implemente a funcionalidade de pesquisar por issues no repositório de vagas do Front-end BR, como mostrado em ui/01-pesquisa.jpg.

Quando um termo for pesquisado, a lista de issues deve exibir vagas abertas que contém o termo (ui/02-pesquisa.jpg). 

Ao clicar no botão "Limpar Pesquisa", a lista de vagas deve voltar ao estado anterior (ui/01-pesquisa.jpg).
