Desafio JavaScript

Básico 03: Operações aritméticas


Escreva uma função que recebe 5 parâmetros (a, b, c, d, e, f).

Faça a soma de a com b.

Subtraia o valor de c.

O resultado da subtração, multiplique por d e faça a divisão por e.

E para finalizar, faça o resultado elevado à potência de f.


Algumas dicas

Você vai precisar dos seguintes operadores:

Adição: 1+2 // resultado 3

Subtração: 3-2 // resultado 1

Multiplicação: 3*3 // resultado 9

Divisão: 9/3 // resultado 3

Expoente: 3**3 // resultado 27


Fique atento à ordem das operações. A próxima operação é sempre utilizando o resultado da anterior.

Dica: pode ser que você precise incluir parênteses.
